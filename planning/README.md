# Planning Phase

This folder contains all documentation related to the planning and requirement analysis of the HealthApp project.

The planning phase includes:

- **Product Vision** – Defines the problem statement, target users, SDG alignment, and overall system goals.
- **User Stories** – Functional requirements written in Agile format.
- **Product Backlog** – Prioritised list of system features.
- **Sprint Plan** – Breakdown of work across Sprint 1 and Sprint 2.
- **Agile Approach** – Explanation of Scrum methodology and tool usage (Trello & GitHub).

This phase establishes the foundation for the system design and implementation stages.
