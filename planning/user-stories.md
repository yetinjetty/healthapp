# User Stories

## Student

1. As a student, I want to register an account so that I can securely access my health records.

2. As a student, I want to log in and log out so that my data remains private.

3. As a student, I want to record my health indicators (BMI, blood pressure, sleep, exercise) so that I can monitor my health.

4. As a student, I want to view a dashboard summary so that I can understand my current health condition.

5. As a student, I want to receive preventive health reminders so that I do not forget screenings.

6. As a student, I want to book a campus clinic appointment so that I can seek medical advice when necessary.

7. As a student, I want to cancel an appointment so that I can manage schedule changes.

## Clinic Staff

8. As clinic staff, I want to view appointment requests so that I can manage bookings.

9. As clinic staff, I want to confirm or reject appointments so that scheduling remains organized.

## Admin

10. As an admin, I want to manage screening schedules so that students always see updated availability.
